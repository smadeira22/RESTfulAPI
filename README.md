# THING RESTful API

## Clone repository

- Open terminal
  - run `git clone `


## Install required dependencies

- On the terminal run:
   - `npm i express cors morgan`
   - `npm i -D nodemon`

## Star API

- run `npm run dev`
- On the web browser open http://localhost:4000/


## Endpoints

- GET /things
- GET /things/:id
- POST /things
- PATCH /things/:id
- DELETE /things/:id
